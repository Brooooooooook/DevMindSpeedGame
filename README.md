# DevMindSpeedGame

A fast-paced mind training game built with C# and .NET.  
Designed to challenge your cognitive reflexes and programming logic.

---

## Features

- Timed speed-based challenges
- Score tracking
- Clean and simple user interface
- Responsive game flow
